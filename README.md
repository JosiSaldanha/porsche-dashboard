# 🏎️ Porsche Sales Dashboard

Dashboard interativo de vendas da Porsche, criado com HTML, CSS e JavaScript puro usando dados sanitizados de uma planilha real.

## 🔗 Acesse o Dashboard

👉 [josisaldanha.github.io/porsche-dashboard](https://josisaldanha.github.io/porsche-dashboard)

## 📊 Funcionalidades

- Filtro por **Estado** (33 estados dos EUA)
- Filtro por **Modelo** (40 modelos Porsche)
- Filtro por **Status de entrega**
- Métricas em tempo real: receita total, ticket médio, % de entregas
- Gráficos de barras: receita por estado, modelos mais vendidos, formas de pagamento
- Tabela completa com 100 registros sanitizados

## 🛠️ Tecnologias

- HTML5 / CSS3 / JavaScript vanilla
- Dados tratados com Python + Pandas
- Deploy via GitHub Pages

## 🎨 Design

Visual inspirado na identidade da marca Porsche: fundo preto, vermelho Racing `#D5001C` e tipografia clean.
